Aplicación de Monitoreo de Canciones

Esta aplicación permite monitorear el rendimiento de las canciones en las plataformas de streaming.

Para ello, la aplicación recopila datos de las siguientes plataformas:

  • Emisora

Los datos recopilados se utilizan para generar informes que muestran el rendimiento de las canciones en cada plataforma.

La aplicación está diseñada para ser utilizada por músicos, sellos discográficos y empresas de marketing musical.
